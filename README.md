# OLDX_MAPPING
在线二维码阵列建图SDK，分别包括树莓派和PC机版本
matlab文件支持对已有地图进行修改包括每一个地标的位置角度，整体地图坐标系位置角度
依赖aruco：2~3
    opencv>2.4.13
    eigen3
